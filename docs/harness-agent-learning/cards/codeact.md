# CodeAct

## 解决什么问题

有些任务用自然语言描述行动很笨重，例如批量统计、解析文件、转换数据。CodeAct 解决的是把行动表达为可执行代码的问题。

## 核心机制

模型生成一段受限代码，Harness 在沙箱或受控环境中执行，收集 stdout、返回值或文件结果，再把观察写回上下文。

## 最小例子

让 Agent 统计一个目录下 Markdown 文件的标题数量。与其反复读文件，可以生成一段脚本遍历文件并输出统计结果。

## 和相近概念的区别

Tool 调用通常是调用预定义函数。CodeAct 让模型临时组合行动逻辑，但安全风险更高，需要权限、超时和文件访问边界。

## 在 Harness Agent 原型中的位置

`execute_codeact` 展示了最小 CodeAct：执行受限 Python 代码并读取 `result`。它只用于学习机制，不是生产级沙箱。
