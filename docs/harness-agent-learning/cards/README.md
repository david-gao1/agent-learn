# Harness Agent Necessary Knowledge Cards

These cards are the first-pass knowledge skeleton for learning Harness Agent engineering. They are intentionally shorter than articles.

Use them before writing long-form pieces:

1. Learn the problem each concept solves.
2. Understand the mechanism.
3. Tie the concept to the minimal prototype.
4. Practice with code or a scenario.
5. Return to article writing only after the concept has an implementation anchor.

Each card follows the same shape:

```md
# Concept

## 解决什么问题
## 核心机制
## 最小例子
## 和相近概念的区别
## 在 Harness Agent 原型中的位置
```
