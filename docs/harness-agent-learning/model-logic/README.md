# Model Logic Notes

Model-logic notes come before prototype practice. They explain the mental model of an Agent pattern so the learner knows what they are about to implement.

Use this structure for each pattern:

```md
# Pattern Name

## 这是什么

Give a plain-language definition.

## 想解决什么问题

Name the uncertainty, failure mode, or control problem.

## 原理是什么

Describe the minimum loop, state transition, or decision rule.

## 如何实践

Give one small scenario, the expected steps, and the acceptance signal.

## 对应原型

Point to the function, test, or demo that implements the idea.
```
