## Shared Editing Checklist

Before expanding any article, check:

1. Can the central question be stated in one sentence?
2. Does every section advance that question?
3. Is any paragraph present only because it is related, not necessary?
4. Is there one running example to reduce reader context switching?
5. What is the one idea the reader should remember?
